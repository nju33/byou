## Usage

```bash
min [command]

Commands:
  min img  Minify *.jpg, *.png, *.gif and *.svg
  min jpg  Minify *.jpg
  min png  Minify *.png
  min gif  Minify *.gif
  min svg  Minify *.svg

Options:
  --version  Show version number                                       [boolean]
  --help     Show help                                                 [boolean]
```
